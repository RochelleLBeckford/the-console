## <p align="center">👩🏾‍💻 JavaScript Outline</p>
-   [ ] The console.log() method:
    -   [ ] Built-in JavaScript funtion that outputs messages
    -   [ ] Used for testing, debugging, and displaying information to developers
    -   [ ] The message to display goes inside parentheses () and is surrounded by single quotes '' or double quotes ""
        ```javascript
            console.log('🌸 Hello World! 🌸')
        ```
-   [ ] Single-line & Mutli-line comments:
    -   [ ] Single-line comments are created using two forward slases ( // )
        -   [ ] Everthing to the right of the ( // )is ignored when the code runs
    -   [ ] Multi-line comments begin with ( \* ) and end with ( */ )
        -   [ ] Everything between the opening and closing symbols is ignored when the program runs
        ```javascript
            // console.log("This won't appear" );

            /*
            This entire section is commented out.
            console.log("This won't run either");
            Neither will this line.
            */
        ```
-   [ ] String concatenation with the '+' operator:
    -   [ ] Joins multiple strings together into one larger string
        -   [ ] The '+' operator combines each piece, and the "" adds spaces between the words
        ```javascript
            console.log('Ancient ' + ' ' + 'Vases' + ' ' + 'of' + ' ' + 'Asia');
        ```
-   [ ] Arithmetic operations in console.log():
    -   [ ] JavaScript can perform mathematical calculations directly
    -   [ ] The result of the calculation is what gets displayed
        ```javascript
            console.log(42 * 3);
        ```
-   [ ] The let keyword for variables:
    -   [ ] Declares a variable that can be reassigned later
    -   [ ] The variable name should describe what it holds
        ```javascript
            //~ For instance:
            let followers = 1000000;
            console.log(followers);
        ```
-   [ ] Creating simple functions:
    -   [ ] Functions are reusable blocks of code that perform specific tasks
    -   [ ] The function keyword begins the declaration
    -   [ ] The return statement specifies what value the function outputs
        ```javascript
            function greetings() {
                return 'Hello, World!';
            }
            console.log(greetings());
        ```
-   [ ] Opening the browser console:
    -   [ ] A feature in Developer Tools where JavaScript code can be typed and run
    -   [ ] Accessible through the browser menus or keyword shortcuts
    -   [ ] Safari short cut:
        -   [ ] opt + cmd + c
        -   [ ] Opens the console directly for typing and runnig JavaScript
            -   [ ] Once open, JavaScript can be typed at the bottom prompt
            -   [ ] Pressing "Enter" runs the code and shows the result
